# Team Task Manager

Projekt i realizuar per detyren `Mini Hackathon - Team Task Manager`.

## Struktura e projektit

- `index.html` - struktura e faqes dhe komponentet kryesore te aplikacionit
- `style.css` - dizajni, responsive layout dhe pamja e listes se detyrave
- `script.js` - logjika per `add task`, `show list`, `delete task` dhe `localStorage`

## Funksionalitetet

- shtimi i nje detyre te re me buton ose me tastin `Enter`
- shfaqja e listes se detyrave ne kohe reale
- fshirja e detyrave nga lista
- ruajtja e detyrave ne `localStorage`
- dizajn i paster dhe i pershtatshem per desktop dhe mobile

## Git dhe GitHub

Ne repo jane perdorur branch-e te ndara per rolet e grupit:

- `feature-html`
- `feature-css`
- `feature-js`

Jane bere commit-e te ndara, merge ne `main`, si edhe nje `merge conflict` real ne `script.js` qe eshte zgjidhur ne versionin final funksional.

## Organizimi i punes ne grup

- Nxenesi 1: HTML dhe struktura e faqes
- Nxenesi 2: CSS dhe dizajni vizual
- Nxenesi 3: JavaScript dhe funksionaliteti

Ky organizim perputhet me kerkesen e detyres per bashkepunim dhe ndarje pune.
